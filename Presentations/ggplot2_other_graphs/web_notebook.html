Boxplots

A boxplot (perhaps known as a box-and-whisker plot to you) is another useful graphic that us statisticians make use of sometimes. In ggplot, boxplots are always oriented vertically, so the variable we wish to construct a boxplot of should be mapped to the y-variable in the aes() function. If you have a single variable that you are graphing a boxplot of, then you should provide the name of the x-axis in the x aesthetic. You could leave this empty by simply stating x = "".
